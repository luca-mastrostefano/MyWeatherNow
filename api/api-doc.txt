#### get
forecast data for Rome

request type: GET

parameters:
    - id: int [optional] if missing, registration is performed.

response: JSON
    - status: int, http response code
    - data : array
        - forecast: array
            - temperature: float, Kelvin scale
            - humidity : int, percent scale
            - wind_speed: float, meters/seconds
            - wind_direction: float, degrees
            - wind_cardinal_direction: string, closest cardinal point over 16 availables.
            - cloudiness: int, percent scale
        - sentence: string, sentence in natural language
        - id: the user ID
